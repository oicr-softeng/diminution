## Development

### Technology
- Node.js 15+
- Express
- SQLite
- Docker

### Build & Run

Install dependencies, then build.
```node
yarn i
yarn build
```

This will compile the source code, and place the output in the `dist/` directory.

To run after it is built:
```node
yarn start
```
